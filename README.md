# Popüler Restoranım

Kullanıcıların şehir ve kategori seçerek restoranları listeleyip puanlayabildiği bir web uygulaması.

## Özellikler
- Şehir ve kategoriye göre restoran listeleme
- Kullanıcı puanlama sistemi
- Admin paneli ile restoran yönetimi (silme/güncelleme)
- Popüler restoranların dinamik olarak sıralanması

## Kullanılan Teknolojiler
- **Backend:** Python, Flask
- **Frontend:** HTML5, CSS3
- **Veritabanı:** MySQL
- **Diğer:** MySQL Connector

## Kurulum
1. Projeyi klonlayın: `git clone https://github.com/azradnlr06/my-popular-restaurant.git`
2. Gerekli kütüphaneleri yükleyin: `pip install flask mysql-connector-python`
3. MySQL'de veritabanını oluşturun
4. `app.py` dosyasını çalıştırın
